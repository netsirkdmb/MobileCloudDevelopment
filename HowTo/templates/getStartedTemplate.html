To make use of the TNSwiftyCheckboxGroup library, you need to have an object that has a variable that is a list and a form that requires multiple selection to set the values of the list in the variable. My app allows the user to enter schools where a school can have multiple types and offer multiple foreign languages. Here are the variables in my School object where you can see that "typeOfSchool" and "foreignLanguagesTaught" are lists. In this guide, we will focus on the "typeOfSchool" input on the form since the methods are the same for the "foreignLanguagesTaught" input.


You will also need to have a form for inputting values to your object. My form for inputting a new school or editing an existing school is shown below. You will notice that instead of including checkboxes for the type of school and the foreign languages, I am using buttons that will take the user to another view where they can make their selections. This is because the TNSwiftyCheckboxGroup is made to work in its own view.


In the view controller document for your form (mine is called SchoolViewController.swift), you will need to connect the buttons that you made for going to the checkbox views to an outlet by using the assistant editor to CTRL-drag each of the buttons to the properties section of your view controller document and give the outlet variables a name. I called mine typeOfSchoolButton and foreignLanguagesTaughtButton.


In order to make the buttons accessible in the view controllers for the checkboxes, you need to also make global variables for the buttons. As you can see in the image above, I have accomplished this by adding the following code above the name of the class. 

    var globalTypeOfSchoolButton: UIButton = UIButton()
    var globalForeignLanguagesTaughtButton: UIButton = UIButton()

The final step for making the buttons accessible in other view controllers is to assign the button outlet variables to the global variables. This is done by adding the following code to the viewDidLoad() function as shown at the bottom of the image above. 

    globalTypeOfSchoolButton = typeOfSchoolButton
    globalForeignLanguagesTaughtButton = foreignLanguagesTaughtButton

Now you are ready to start writing the code for the view controller that you are going to use for the checkboxes.

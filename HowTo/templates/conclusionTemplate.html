If you have followed the steps up to this page, your TNSwiftyCheckboxes should work, but I wanted to go over a few more details to give your app a little more polish.


Text on "Choose Type(s) of School" Button

When we replace the "Choose Type(s) of School" text on this button with the types of schools that the user has checked, the list can get long and go off of the screen. To make it look cleaner, you can choose "Truncate Tail" in the Line Break dropdown of the Attribute Inspector for the button. This will truncate the text on the button so that it fits on the screen and add an elipsis (...) at the end to idicate that there is more text that doesn't fit on the screen.


Image Checkboxes

If you choose to use the image checkbox type like I did, you must provide the images for the checkboxes. The images must be 40x40 pixels in size and you need to provide an image for an unchecked box and for a checked box. I used the following images, which I got from the sample TNSwiftyCheckboxGroup project that can be obtained by cloning the repo on the Github page.



You can add the images to your project by clicking on Assets.xcassets in the list of files in your project. At the bottom of the column that appears on the left side of the center screen, click the "plus" symbol. Click "New Image Set" in the menu that appears. Rename the image set to "checked-image" and drag your image file for a checked box to the outline above the 2x label. Follow the same steps for your image file for an unchecked box, renaming the image set to "unchecked-image".

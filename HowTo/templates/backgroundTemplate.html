I started out thinking that I would do this assignment on how to make an iOS app using Swift. I had trouble figuring out how to do all of the form input things that I needed to do for the Mobile Development assignment and ended up making a mobile web app instead. Therefore, I thought this was a good chance to learn how to make a smaller and slightly simpler app that would allow me to try to figure out some of the form inputs that had given me trouble before. I decided to make an app to display and edit the simplest table in my API (the schools table).

The main aspect of form input that had given me trouble for the Moblie Development assignment was how to allow a user to select multiple options from a list of choices (like checkboxes on a website). The built-in options don't seem to allow multiple selection. So, when I was working on my new iOS app, I started looking around on the internet for libraries that I could use for this functionality. While I was searching, I stumbled upon a library called TNSwiftyCheckboxGroup. This seemed library like exactly what I needed, if I could only figure out how to use it! There is documentation, but it is very vague and incomplete.

This is how my How-To idea was born. In this guide, I'm going to show how to use the "TNSwiftyCheckboxGroup" for form input. I will assume some basic knowledge of Swift and using the storyboard to make layouts in Xcode.

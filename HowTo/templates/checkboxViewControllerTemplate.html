We will start out with a fairly simple view controller that we will modify again after we create our storyboard.

Make a new ViewController file:

  1. Make a new UIViewController file by clicking New->File in the Xcode File menu.
  2. In the dialog that shows up, choose Source under iOS on the left side and then pick Cocoa Touch Class before clicking the next button.
  3. In the class field of the dialog, give your new class a name. Mine is called TypeOfSchoolViewController. The "ViewController" suffix is added automatically when you select UIViewController in the "Subclass of" dropdown.
  4. Make sure to check the box next to "Also create XIB file" before clicking the next button again.
  5. Navigate to the folder with all of your other .swift files for this project and click the create button.

Add Code to the New ViewController File

Now it is time to add some code. Your new view controller file will initialize with a viewDidLoad() and a didReceiveMemoryWarning() function as well as a prepareForSegue function that has been commented out. First, we want to make it so we can use TNSwiftyCheckboxGroup functionality in this view controller by adding an import statement import TNSwiftyCheckboxGroup below the import UIKit line at the top of the file. Next, add the following lines above the viewDidLoad() function. 

    // MARK: - IBOutlets
    
    // MARK: - Properties
    var checkboxVC: TNSwiftyCheckboxViewController!
    var typeOfSchoolString: String = String()

The commented lines starting with // MARK: are used by Xcode to help you jump to different parts of your file.


The IBOutlets section will be used later when we connect this view controller file to the storyboard. The variables that we just added to the Properties section will also be used later when we tell the view controller what to do with the information in the checkboxes as we navigate to and from the page in the prepareForSegue function. We have enough in this file now to start on the storyboard.


Now you are ready to start adding the views for the TNSwiftyCheckboxGroup to the storyboard!
